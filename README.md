# Laboratorio de Persistencia y Despliegue

Repositorio de prácticas en Java orientado a la experimentación con
persistencia local, gestión de logs y despliegue de aplicaciones ejecutables.

---

## ¿Qué aborda este proyecto?

- **Gestión de entorno**
    - Validación y creación automática de estructuras de directorios
    - Preparación de rutas de almacenamiento locales de forma segura

- **Robustez y resiliencia**
    - Uso de `try-with-resources` para una correcta gestión de recursos
    - Sistema de doble log (consola y archivo físico) para trazabilidad y diagnóstico de errores

- **Despliegue**
    - Generación de artefactos ejecutables (`.jar`)
    - Preparación para ejecución en entornos de servidor o línea de comandos

---

## Tecnologías utilizadas

- Java SE
- File I/O
- Java Time API
- CLI (Command Line Interface)
